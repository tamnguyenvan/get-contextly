How it works

1

Configure your API Key

Contextly is BYOK (Bring Your Own Key). Get a free API key from Google AI Studio and enter it in the extension settings. This ensures your usage is private and free.

2

Select any text

Highlight a sentence, a paragraph, or an entire email body on any website.

3

Choose your action

A small floating menu will appear. Choose to fix grammar, translate, or summarize. The result appears in a clean overlay.

Frequently Asked Questions

Is Contextly free to use?
Yes! The extension itself is free and open-source. However, it requires a Google Gemini API Key. Google provides a generous free tier for the Gemini API that is sufficient for most personal use cases.
Where is my data sent?
Your selected text is sent directly from your browser to Google's Gemini API servers for processing. We (the developers) do not collect, store, or see your data.
How do I get an API Key?
You can generate an API key for free at Google AI Studio. Once you have it, paste it into the Contextly extension settings.
